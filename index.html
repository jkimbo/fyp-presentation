The Problem

  • No way of knowing when your next coach is going to arrive
  • Coaches travel longer distances
  • Delays can be significant

The Solution

Coach+

  • Tracks coaches in real time and predicts what time they are going to arrive at a stop
  • Web client can access this data and show it to the user
  • Easy and intuitive interface
  • A platform on which to build an ecosytem of applications

The Product

  • Easy to use
  • Extensible
  • Modular
  • Fully customisable
  • Node.js
  • Websockets
  • HTML5

Get coaches near a stop

    
      GET /coaches

Parameters:

    
      {
        stop: [stop id],
        route: [route id - optional]
      }

Response:

    
      {
        coaches: [
          {
            id: 101,
            distance: 1801,
            duration: 193,
            location: {
              lat: 51.49466,
              lng: -0.19517,
              timestamp: "1340641857111"
            },
            route: {
              id: 69,
              points: [
                ...
              ]
            }
          },
          {
            ...
          }
        ]
      }

Post new coach location

    
      POST /coach/:id

Parameters:

    
      {
        lat: [latitude],
        lng: [longitude],
        timestamp: [timestamp]
      }

Response:

    
      { 
        message: Success!
      }
