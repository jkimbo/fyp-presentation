Get coaches near a stop

    
      GET /coaches

Parameters:

    
      {
        stop: [stop id],
        route: [route id - optional]
      }

Response:

    
      {
        coaches: [
          {
            id: 101,
            distance: 1801,
            duration: 193,
            location: {
              lat: 51.49466,
              lng: -0.19517,
              timestamp: "1340641857111"
            },
            route: {
              id: 69,
              points: [
                ...
              ]
            }
          },
          {
            ...
          }
        ]
      }

Post new coach location

    
      POST /coach/:id

Parameters:

    
      {
        lat: [latitude],
        lng: [longitude],
        timestamp: [timestamp]
      }

Response:

    
      { 
        message: Success!
